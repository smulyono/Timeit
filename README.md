# Timeit README

Simple task timer which allows setup a timer while doing specific task.


## Features

* `Start new Task`
* `Stop / Pause Timer`
* `Task completion with completion time`, show how long does task took.
* `Timer configuration` 
    
    * __timeit.duration__, timer duration for individual task
